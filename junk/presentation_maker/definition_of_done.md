---
title: "Definition of Done"
description: "Checklist of criteria for a complete presentation"
---

- [ ] All user-provided content is represented — nothing important omitted
- [ ] Zero actionable lint findings (text-overflow, canvas-overflow, low-contrast)
- [ ] All generated images render correctly
- [ ] Bundled HTML (index.bundle.html) opens correctly in a browser
- [ ] Fonts load correctly
- [ ] Theme is applied consistently across all slides
- [ ] Layout variety — not every slide uses the same pattern
- [ ] User has approved the exemplar slides (style lock)
- [ ] User has approved the final presentation (review phase)
- [ ] Manifest status shows all slides complete
- [ ] All git commits made per the archival policy
- [ ] PDF export produced and verified
