# SlideKit Migration Guide

> This guide helps AI agents and humans update presentation code when upgrading between breaking SlideKit versions. If you see a version compatibility error from `init()`, follow the relevant section below.

## How to Use This Guide

When `init({ minVersion: '...' })` reports a compatibility error, find the section matching your upgrade path below. Each section lists:
- What changed and why
- Code patterns to find and replace
- Before/after examples

An AI agent can read this guide and automatically apply the necessary changes to your presentation code.
